# Project for BFU
**Игра Pong**
- Требования:
1. standart C++ 17+
2. Cmake version 3.15+
3. Клавиатура
 -  **Управление**
1. Первый игрок(слева) **W - ввeрх**, **S - вниз**
2. Второй игрок(справа) **Up - ввeрх**, **Down - вниз**
3. **Space** - продолжить игру
- Использоваллась среда разработки CLion
- библиотка SFML
- Компилятор MinGW
